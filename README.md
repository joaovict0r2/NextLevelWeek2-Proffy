## Como utilizar? 

### Clone
- Clone este repositório na sua máquina local
```
git clone https://github.com/jpgSouza/next-level-week2.git
```

### Instalação
- Na pasta do projeto, rode o comando no seu terminal
```sh
npm run dev
```

### Navegador
- Porta utilizada
```sh
localhost:5500
```
